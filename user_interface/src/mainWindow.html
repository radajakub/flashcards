    Help

    Card = pair of texts, only one is known at the time

    Tag = group of cards, unit for testing

    Create card

    • Adds new card into your collection
    • Front side text field fill with text to be shown on the front side of the flashcard
    • Back side text field fill with text to be shown on the back side of the flashcard
    • Select any of the displayed tags to add this card to
    • Click Save to save the card
    • Click Cancel to discard changes

    Create tag

    • Adds new tag into your collection
    • Tag name text field fill with text to be treated as a name of the tag
    • Click Save to save the tag
    • Click Cancel to discard changes

    Edit card

    • Shows list of all cards in your collection with details about each card
      • Front and Back side and Number of tags which contain the card
      • Edit button - open form for editing selected card
      • Delete button - remove card from collection
    • Editing is same as creating - fields and checkboxes are filled with current information about selected card
    • Save to save changes
    • Cancel to discard changes

    Edit tag

    • Shows list of all tags in collection with details about each tag
      • Tag name, Number of cards which are in the tag and Success rate of the last test
      • Edit button - open form for editing selected tag
      • Delete button - remove tag from collection
    • Only Tag name can be changed - field filled with current name
    • Save to save changes
    • Cancel to discard changes

    Tests

    • Choose tag to test from listed options
    • Choose type of test (browse, choices, write) and select if card will be tested normally or with Front and Back swapped (Reversed switch)
    • Click Back to tag selection to return to selection of tag to test

    Browse

    • No questions in this type of test
    • Browsing cards by clicking Previous or Next buttons
    • Clicking on card flips it to show the hidden side
    • On top of the page is status bar on which is shown which card from how many is shown
    • Click Back to test selection to return back to selecting type of test

    Choices

    • On the top of the page is shown Front side of the card
    • Below are listed 4 choices from which Only one is correct
    • Select correct one by clicking on them
    • Click on Check answer to confirm answer - page with result - shows whether the answer was correct or not, when not it provides correct one, too
    • Click on Next question to move to next question
    • On the right side are three status bars
      • Number of correct answers, wrong answers and answers to be answered
    • After the last question a summary is shown
      • Percentage of correct answers is shown and comparison with previous test
      • Click on Show answers to display a table with all answers and correct answers
      • Click on Continue to test selection to finish test and return to selecting another test

    Write

    • On the top of the page is shown Front side of the card
    • Below is a field where to put answer
    • Click on Check answer to confirm answer - page with result - shows whether the answer was correct or not, when not it provides correct one and highlights mistakes in given answer
    • Click on Next question to move to next question
    • On the right side are three status bars
      • Number of correct answers, wrong answers and answers to be answered
    • After the last question a summary is shown
      • Percentage of correct answers is shown and comparison with previous test
      • Click on Show answers to display a table with all answers and correct answers
      • Click on Continue to test selection to finish test and return to selecting another test

    Import

    • Only one button - show file system
    • Select file to import (it has to be .csv format)
    • If the file is written correctly, data will be added to collection
    • (If imported card already exists in collection, it will be skipped)

    Export

    • One text field - write name of file in which data will be exported (letters and numbers only)
    • File will be saved to export folder in the main folder of this app
